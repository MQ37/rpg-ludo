# rpg-ludo
School project

### How to run
- `make run` or `python3 main.py`

### How to test
- `make test` or `python3 tests/tests.py`

### How to docker
- **Build image**
  - `docker build -t mq37/rpg-ludo .`
- **Run the image**
  - `docker run -it --name ludo mq37/rpg-ludo`
- **Docker hub**
  - https://hub.docker.com/r/mq37/rpg-ludo
